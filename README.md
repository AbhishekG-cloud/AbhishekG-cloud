### 🔹 Bayesian A/B Testing Framework

A production-ready framework for Bayesian online experimentation that compares **Frequentist** and **Bayesian** approaches to A/B testing.

**Highlights**

* Bayesian inference using Beta-Binomial conjugate models
* Monte Carlo simulation for decision making
* Sequential experiment updates
* Business metrics (Expected Loss, Expected Uplift, Probability to Beat Control)
* Interactive Streamlit dashboard
* Modular Python package with reusable components

**Tech Stack**

Python • NumPy • SciPy • Plotly • Streamlit • Bayesian Statistics
